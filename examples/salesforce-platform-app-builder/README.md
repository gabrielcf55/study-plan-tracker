# 🚀 Exemplo · Salesforce Platform App Builder

Plano **iniciante**, partindo do zero, para a **Salesforce Certified Platform App Builder**.

- **Perfil:** iniciante, ~1h/dia.
- **Duração:** 8 semanas de estudo + semana do exame.
- **Exame:** 65 questões, 105 min, nota de corte 63% (taxa US$ 200; retake US$ 100).
- **Domínios:** Business Logic & Automação (28%), Fundamentos (23%), Data Modeling (22%), User Interface (17%), App Deployment (10%).
- **Mocks:** simulados de múltipla escolha, com bancos gratuitos (Salesforce Ben, ExamTopics, Practice Test Geeks) e opções pagas.

Aplicar este plano na raiz `plan/`:

```bash
npm run use-example -- salesforce-platform-app-builder
```

Depois, ajuste em `plan/` conforme [`plan/SCHEMA.md`](../../plan/SCHEMA.md). Dados de custo e formato conferidos em jun/2026 — confirme sempre na fonte oficial.
