# 🏆 Exemplo · Salesforce CTA

Plano para a **Salesforce Certified Technical Architect (CTA)** — a credencial mais alta do ecossistema Salesforce.

- **Perfil:** profissional Salesforce full-time, com pouco tempo livre por dia.
- **Duração:** ~18–20 meses (≈ 80 semanas), em 6 fases.
- **Etapas:** Architect Evaluation (US$ 1.500) → Review Board Exam (US$ 4.500).
- **Mocks:** cenários de arquitetura (Evaluation 60 min · Review Board 180 min), com referências a comunidades e sessões de mock review.

Aplicar este plano na raiz `plan/`:

```bash
npm run use-example -- salesforce-cta
```

Depois, ajuste em `plan/` conforme [`plan/SCHEMA.md`](../../plan/SCHEMA.md). Dados de custo e formato conferidos em jun/2026 — confirme sempre na fonte oficial.
