# 📄 Exemplo · Template

Esqueleto **neutro** que demonstra todos os campos do schema, sem conteúdo específico. É o ponto de partida para montar um plano novo — e o mesmo conteúdo que vem na raiz `plan/` por padrão.

- **Fases:** 5 (Orientação → Fundamentos → Aprofundamento → Revisão & Simulados → Exame), em 12 semanas.
- **Conteúdo:** textos de exemplo marcados com “(substitua)”, mostrando onde editar.

Aplicar (ou voltar a raiz `plan/` ao estado inicial):

```bash
npm run use-example -- template
```

Para preencher: edite `plan/` seguindo [`plan/SCHEMA.md`](../../plan/SCHEMA.md), ou rode a skill `/plan-builder` para montar por entrevista.
